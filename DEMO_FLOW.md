# DEMO_FLOW.md

## Demo Goal
Show `Playground` as a **content intelligence tool** for Indonesian creators/affiliate marketers.

## Demo Flow v1
1. Open landing page
2. Explain product focus:
   - viral hooks
   - captions/scripts
   - hashtags
   - sounds
   - engagement
   - transcripts
3. Run TikTok search for a keyword
4. Show enriched result fields:
   - hook
   - content
   - caption
   - metrics
5. Run profile scrape
6. Run comments scrape on a valid TikTok video
7. Download JSON/CSV output

## Demo Notes
- Prefer TikTok for stable demos
- Avoid dead/private TikTok URLs for comments demo
- Do not pitch as revenue estimator or creator analytics tool
- Position as public content research tool / content pattern discovery tool
